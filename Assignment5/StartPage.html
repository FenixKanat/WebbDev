CSS Info

Inline CSS används genom att placera CSS-kod direkt i HTML-elementet med hjälp av style-attributet.
Det är enkelt att implementera och snabbt att applicera på enskilda element, men kan göra koden rörig och svår att underhålla.
Externa stilmallar skapas som separata .css-filer och länkas till HTML-dokumentet med hjälp av link-elementet.
De separerar presentation från innehåll, gör koden renare och enklare att underhålla.
Stilmallarna kan återanvändas på flera sidor och ger bättre struktur och organisering av CSS-regler.
