Frågor och svar om CSS

Vad är CSS?

CSS står för Cascading Style Sheets och är ett språk som används för att beskriva hur HTML-element ska presenteras på en webbsida. Genom att använda CSS kan man ändra utseendet och layouten på en webbsida, inklusive färger, typsnitt, storlekar, placeringar och mycket mer.

Varför finns CSS?

CSS finns för att separera utseendet från innehållet på en webbsida. Genom att separera HTML (innehållet) och CSS (utseendet) blir det lättare att underhålla och uppdatera webbsidor. CSS möjliggör även en enhetlig design på en webbplats genom att använda samma stilmallar på flera sidor.

Hur använder vi oss av CSS?

Man kan använda CSS på flera olika sätt. Det vanligaste sättet är att skapa en separat CSS-fil med stilregler och sedan länka till den från HTML-dokumentet genom att använda link-elementet. Till exempel kan man skapa en fil med namnet "style.css" och inkludera följande kod i HTML-filen:

          
          <link rel="stylesheet" href="style.css">

I CSS-filen kan man sedan definiera olika stilregler för olika HTML-element. Till exempel kan man använda selektorer för att välja specifika element och tillämpa olika egenskaper och värden på dem. Exempelvis kan man använda följande kod för att ändra bakgrundsfärgen på alla <p>-element:

          
          p {
              background-color: #f2f2f2;
          }

På detta sätt kan man skapa och anpassa stilmallar som appliceras på olika element på webbsidan.

Det finns olika versioner av CSS: Vilka versioner finns det?

Det finns flera versioner av CSS, inklusive CSS1, CSS2, CSS2.1, CSS3 och CSS4 (som ännu inte är helt färdigställd). Varje version har introducerat nya funktioner, egenskaper och förbättringar jämfört med tidigare versioner.

Varför finns det olika versioner? (Varför är det inte bara en?)

Det finns olika versioner av CSS eftersom webbtekniken och behoven förändras över tiden. Genom att introducera nya versioner kan man lägga till nya funktioner och egenskaper som gör det möjligt att skapa mer avancerade och interaktiva webbsidor.
